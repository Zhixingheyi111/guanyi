# 数据来源声明

观易 App 所用经典原文、翻译、注释的来源记录。**每条数据可追溯。**

## 版权策略

### 原文：公版优先
- 仅使用公元 **1923 年前**的作品（中国大陆+港澳台版权法的稳定 public domain 阈值）
- 优先标准学术通行本（如朱熹章句、王弼注本、玄奘译本等）
- 用 Chinese Text Project (ctext.org) / 中国哲学书电子化计划 / 中文维基文库 等公版数字资源校对

### 翻译：自录
- **不直接抄现代译者**（杨伯峻《论语译注》、陈鼓应《老子今注今译》等都受版权保护）
- 基于原文 + 多家古注（公版）+ 上下文，自己写白话翻译
- 标注 "自译" 或 "基于 朱熹《xxx集注》"

### 注释：自录 + 公版古注引用
- 字词义引用公版古注（朱熹卒 1200 AD、王弼卒 249 AD、郭象卒 312 AD 等都是 700+ 年前作品）
- 现代理解写自己的话
- 不抄现代学者书的注释

### AI 解读
- 实时生成，UI 上明确标 "AI 生成"
- 不固化进数据库（保持事实/解读分离原则）

## 二期数据来源（Phase 2：9 部经典）

**通则**：每部书三层来源
1. **原文** = 公版标准本（>1923 年前作品）

3. **注释** = 自录 + 引用公版古注

### A1. 论语 `src/data/classics/lunyu.js`
- **原文版本**：朱熹《论语集注》（南宋，1177 AD 成书，公版）通行本
  - 校对参考：ctext.org/analects、中文维基文库《论语》
- **结构**：20 篇 × ~25 章 = 约 500 章节
  - 篇名：学而、为政、八佾、里仁、公冶长、雍也、述而、泰伯、子罕、乡党、先进、颜渊、子路、宪问、卫灵公、季氏、阳货、微子、子张、尧曰
- **翻译**：自译，参考朱熹《集注》义理 + 上下文
- **注释**：自录字词义；引用《集注》关键义理（标 "朱注"）
- **persona**：默认孔子（confucius）；可选子夏、颜回、曾子等弟子视角

### A2. 孟子 `src/data/classics/mengzi.js`

- **结构**：7 篇上下 = 14 卷，约 260 章
  - 篇名：梁惠王、公孙丑、滕文公、离娄、万章、告子、尽心
- **翻译**：自译
- **注释**：自录 + 朱注引用
- **persona**：孟子（mengzi，新增）；朱熹（既有）
