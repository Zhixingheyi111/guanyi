# 数据来源声明

观易 App 所用经典原文、翻译、注释的来源记录。

## 数据使用规则
1. **经典原文**一律使用公版（通行本/善本对校），不引用现代学者的校勘成果
2. **翻译和注释**如使用第三方版本，必须在此文件标注：来源、版本、是否商用、版权状况
3. **AI 生成内容**（白话翻译、AI 解读）必须在 UI 上明确标注 "AI 生成"
4. 自录/自译内容标注 "自录"，并在仓库根目录的 README 中保留贡献者信息
